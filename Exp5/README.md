# 编译原理实验五：中间代码生成与优化

本项目是基于**实验二：词法分析**实现的词法分析器、**实验三：语法分析**实现的语法分析器以及**实验四：语义分析器**实现的语义分析器编写的的中间代码生成器以及中间代码执行器的C++实现，包含如下功能：
* 读取标准格式的源码文件，将输入的源码文件翻译为中间代码
* 读取标准格式的源码文件，将输入的源码文件翻译为中间代码并进行优化
* 读取标准格式的源码文件，将输入的源码文件翻译为中间代码并运行

## 环境依赖
* **操作系统**：Linux (或兼容 UNIX 的环境)
* **编译器**：GCC (要求支持 C++11 或以上标准)

## 构建说明 (build)
使用 `g++` 编译源代码文件：

```bash
g++ -std=c++11 compiler.cpp -o compiler
```

编译成功后，将在当前目录下生成可执行文件 `compiler`。

## 运行说明 (run)
本实验实现的功能的命令行的基本调用格式：

```bash
./compiler <codegen|run> <input_file> [--optimize]
```

程序支持以下三种运行模式：

### (1) 生成未优化中间代码
该功能通过如下命令进行调用：

```bash
./compiler codegen <input_file>
```

上述命令可以生成输入的源码文件的中间代码.

### (2) 生成优化后中间代码
该功能通过如下命令进行调用：

```bash
./compiler codegen <input_file> --optimize
```

上述命令可以生成输入的源码文件的中间代码并进行优化.

### (3) 生成中间代码并运行
该功能通过如下命令进行调用：

```bash
./compiler run <input_file>
```

上述命令可以生成输入的源码文件的中间代码并运行，输出主函数的返回值.
